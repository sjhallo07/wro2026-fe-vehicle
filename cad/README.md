# CAD Files

This directory contains 3D-printable STL files and editable source files for
the vehicle chassis and mechanical components.

## Files

| File | Description |
|------|-------------|
| `chassis_main.stl` | Main chassis frame (280 mm × 180 mm) |
| `motor_mount.stl` | Rear DC motor mounting bracket |
| `servo_bracket.stl` | Front servo and steering linkage bracket |
| `wheel_hub.stl` | Hub adapter for 65 mm wheels |

## Printing Recommendations

- **Material:** PETG or PLA+ (PETG preferred for durability)
- **Layer height:** 0.2 mm
- **Infill:** 40 %
- **Supports:** Required for motor mount and servo bracket

Editable source files (FreeCAD / Fusion 360) will be added when available.
